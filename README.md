# Beastie
Customer Management System for Pet shop

# GetStarted
##### Requirement
node
npm
mongoDB

##### Installation
```
npm run install:all
```
##### Development
```
npm run dev:front
npm run dev:back
```
##### Production
```
npm run beastie
```

##### Templates
```
/client/src
gulp module --name AngularModuleName --parent ParentFolder
gulp component --name AngularModuleName --parent ParentFolder

/routes
gulp route --name RouteName --parent ParentFolder
```
